# Housing Price Analysis using SQL & Python

## Objective
Analyze housing price data to understand how different features impact property prices.

## Tools Used
- SQL Server
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Price vs Area
- Bedrooms, Bathrooms, Parking impact
- Furnishing status analysis

## Key Insights
- Area and bathrooms have strong influence on price
- Furnished houses are priced higher
- Parking and AC increase house value
